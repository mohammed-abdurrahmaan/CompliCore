Phase Summary

🏢 Onboarding

Account creation, company setup wizard, EIN, controlled groups, integrations, seasonal/TRICARE flags

Entry Point

🔗 Step 1: Payroll Integration

API connect (ADP) or CSV upload. Auto-ingest hours, wages, classifications monthly. PTO mapping, error flagging.

Data Ingestion

📊 Step 2: ALE Dashboard

ALE/Non-ALE status, 12-month FTE trailing average, threshold alerts (red/yellow/green).

Status Monitor

👥 Step 3: Classification

Look-Back or Monthly method. Measurement/admin/stability periods. FT/PT designation table.

Workforce

📋 Step 4: Offer Tracking

Mark coverage offered/not monthly. Real-time offer %, dependent coverage tracking. ≥95% compliance check.

Compliance

📬 Step 4a: Open Enrollment

HR opens window → employees notified → accept/decline with tier selection → 48hr reminders → HR approval → proof PDFs → IRS codes recalculate.

Enrollment

💰 Step 5: Affordability

W-2, Rate of Pay, FPL safe harbors. Per-employee calculation. Red/yellow/green status bars.

Testing

✅ Step 6: MEC & MV

7-point MEC check, HHS MV calculator. MV fail → Actuary Marketplace → AI matching → quote → certification → validation.

Validation

⚠️ Step 7: Penalty Simulation

CFO view with dynamic 4980H(a)/(b) calculator. Scenario toggles for workforce/premium changes.

Risk

📄 Step 8: IRS Filing

1095-C/1094-C auto-generation. PDF download, AIR XML export, or in-app e-file. Deadline banners, rejection handling.

Filing

📨 Step 9: Notice Response

Upload Letter 226J → OCR parse → auto-match employees → draft response → submit → track resolution.
